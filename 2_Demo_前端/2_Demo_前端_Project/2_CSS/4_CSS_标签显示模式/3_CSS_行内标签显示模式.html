3_CSS_行内标签显示模式

行内元素(内联元素)，不占有独立的区域，仅仅靠自身的字体大小和图像尺寸来支撑结构，一般不可以设置宽度高度和对齐属性，常用于控制页面文本样式

常用的行内元素有：

  1. a
  2. strong
  3. b
  4. em
  5. i
  6. del
  7. s
  8. u
  9. ins
  10. span
其中span是最典型的行内元素

行内元素的特点：

  1. 和相邻行内元素在同一行内
  2. 高、宽无效，但是可以控制padding，margin，垂直方向无效
  3. 默认宽度就是内容本身的宽度
  4. 行内元素只能容纳文本或其他行内元素
