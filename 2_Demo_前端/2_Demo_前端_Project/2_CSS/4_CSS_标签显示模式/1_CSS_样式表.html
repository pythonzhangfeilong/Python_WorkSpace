1、行内式样式表(内联样式)

内联样式表，也就是在标签内部直接写样式属性,适用于小部分样式修改调整

行内样式实例:

<div style="属性值" >行内样式</div>

2、内部样式表

内部样式表，也就是最常规的写法，把样式写在head标签中

注意：由于使用的html5，所以style标签可以写type="text/css"也可以不写，根据需要选择就行

内部样式表实例

<head>
<style>
选择器{属性1:属性值1 属性2:属性值2}
</style>
</head>

外部样式表(外链式)

嵌入式是将所有的样式放在一个或多个以CSS为拓展名的外部样式文件中，通过link标签将外部样式表文件链接到html文件中

基本语法如下:

<link href="css文件路径" type="text/css" rel="stylesheet" >
