一、CSS_行间距

line-height属性用于设置行间距就是行与行之间的距离，也就是字体与字体之间的垂直距离，一般称为行高

line-height常用属性值单位有3种：像素px、相对值em、百分比%，像素使用的最多，最常用的行间距是7.8px

二、CSS_水平对其方式

text-align属性用设置文本的水平对齐方式

  1. left 左对齐(默认)
  2. right 右对齐
  3. center 居中

三、CSS_首行缩进

text-indent属性用于设置文本首行缩进，其属性值可以为不同单位的数值、em字符宽度的倍数、相对与浏览器窗口的百分比允许使用负值，建议使用em作为单位

1em就是一个汉字的宽度
