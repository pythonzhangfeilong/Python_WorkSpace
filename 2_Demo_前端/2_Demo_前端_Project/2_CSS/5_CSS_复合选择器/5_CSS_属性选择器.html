CSS_属性选择器

属性选择器：选取标签带有某些特殊属性的选择器

属性选择器用[]来表示

属性选择器名称 属性选择器语法 解释
属性选择器 选择器[标签属性]{属性1:属性值1; 属性2:属性值2;} 标签存在的属性
属性=属性值选择器 选择器[标签属性=标签属性值]{属性1:属性值1; 属性2:属性值2;} 标签属性完全等于标签属性值
包含属性值val开头选择器 选择器[class^="val"]{属性1:属性值1; 属性2:属性值2;} 属性值里面包含val在开始的位置
包含属性值val结束选择器 选择器[class$="val"]{属性1:属性值1; 属性2:属性值2;} 属性值里面包含val在结束的位置
包含属性值val任意位置选择器 选择器[class*="val"]{属性1:属性值1; 属性2:属性值2;} 属性值里面包含val在任意的位置
