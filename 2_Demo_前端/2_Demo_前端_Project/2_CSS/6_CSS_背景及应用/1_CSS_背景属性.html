CSS_背景属性：background

属性名 解释 属性值选择
backgroubd-color 背景颜色
background-image 背景图片地址
background-repeat 是否平铺
  1. repeat，默认值，背景在水平和垂直方向上平铺
  2. repeat-x，背景在水平方向上平铺
  3. repeat-y，背景在垂直方向上平铺
  4. no-repeat，背景不平铺
  5. inherit，规定应该从父元素继承 background-repeat 属性的设置
background-position 背景位置
    方位名词写法:
  1. top上
  2. bottom下
  3. left左
  4. right右
  5. center中
    准确值写法:
  1. x值(水平方向)，y值(垂直方向) 准确值
    准确值+方位名词写法:
  1. x值(水平方向) center
  2. y值(垂直方向) center
background-attachment 图片固定还是滚动
  1. fixed，背景固定
  2. scroll，背景不固定，默认值
  3. inherit，规定应该从父元素继承 background-attachment 属性的设置
背景的复合写法 background:背景颜色 背景图片地址 背景平铺 背景滚动 背景位置
