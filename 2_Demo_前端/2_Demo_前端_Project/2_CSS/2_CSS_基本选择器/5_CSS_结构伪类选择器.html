CSS_结构伪类选择器

  1. :first-child 选取属于父类元素的首个子元素的指定选择器
  2. :last-child 选取属于父元素的最后一个子元素的自定选择器
  3. :nth-child(n) 匹配属于父元素的第n个子元素，不论元素的类型，even偶数，odd奇数，n从0开始
  4. :nth-last-child(n) 选择器匹配属于元素的第n个子元素的每个元素，不论元素的类型，从元素的最后一个子元素计数，n可以是数字、关键词或公式

:nth-child(n) 正数

:nth-last-child(n) 倒数，与 :nth-child(n)正好相反

  • 第2个孩子
