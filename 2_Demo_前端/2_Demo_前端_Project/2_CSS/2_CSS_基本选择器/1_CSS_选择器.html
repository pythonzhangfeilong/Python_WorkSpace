CSS_选择器

一、标签选择器(元素选择器)

标签选择器:是指用HTML标签名称作为选择器，按标签名称分类

  1. 标签名{属性1:"属性值1"; 属性2:"属性值2"}
  2. 元素名{属性1:"属性值1"; 属性2:"属性值2"}

标签选择器的最大有点就是可以一次性选择到页面所有的需要标签

二、类选择器

给类名命名时也是采用见名知意的方式，类名可以是多个，用空格分割，不建议采用纯数字和汉字

类选择器可以多次重复使用

类选择器使用"."(英文点号)进行标示，后面紧跟类名

  1. .类名{属性1:"属性值1"; 属性2:"属性值2"}

在标签中添加class="类名"

三、id选择器

id选择器唯一不可重复

类选择器使用"#"(英文井号)进行标示，后面紧跟id名

  1. #id名{属性1:"属性值1"; 属性2:"属性值2"}

四、通配符选择器

使用英文状态下"*"，可以选择页面中的所有的标签

  1. *{属性1:"属性值1"; 属性2:"属性值2"}
