1、表单控件input（输入）

通过type属性来控制表单类型

属性 属性值 描述
type text 单行文本输入框
password 密码输入框
radio 单选按钮
checkbox 复选框
button 普通按钮
submit 提交按钮
reset 重置按钮
image 图片形式的提交按钮
file 文件域
name 用户自定义 控件的名称
input 用户自定意 input控件中的默认文本值
size 正整数 input控件在页面中的显示宽度
checked checked 定义控件默认被选中的项
maxlength 正整数 控件允许输入的最多字符数
