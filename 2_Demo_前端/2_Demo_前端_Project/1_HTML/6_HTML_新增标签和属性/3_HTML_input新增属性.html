属性 用法 含义
placeholder <input type="text" placeholder="请输入用户名" > 占位符提示输入框可输入的信息
autofocus <input type="text" autofocus > 规定页面加载时，input自动获得焦点
multiple <input type="file" multiple > 多文件上传
autocomplete <input type="text" autocomplete="off" > 规定表单是否启用自动完成功能，有俩个值，一个是off，一个是on，off代表不记录输入的值，on代表记录输入的值
required <input type="text" required > 必须输入
accesskey <input type="text" accesskey > 规定激活（使元素获得焦点）元素的快捷键，采用alt+字母
